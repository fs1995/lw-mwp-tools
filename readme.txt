View resource usage, access and error logs, and more on the Liquid Web Managed WordPress Hosting Platform.

== Description ==

View your PHP error log, NGINX logs, and view system resource usage on the Liquid Web Managed WordPress Hosting Platform without having to contact support.

This is an unofficial plugin and is not affiliated with Liquid Web. For any bug reports or suggestions, let me know in the plugins support forum.
