Easy access to system logs and resource usage on the Liquid Web Managed WordPress Hosting Platform.

== Description ==

This plugin allows easy access to your PHP error log, nginx logs, and to view system resources without having to contact support.

This is an unofficial plugin and is not affiliated with Liquid Web.

== Installation ==

= MINIMUM REQUIREMENTS =

* Liquid Web Managed WordPress V2 platform

= AUTOMATIC INSTALL =

From your WordPress dashboard, navigate to the Plugins page and click Add New. Search for the "LW MWP Tools" plugin, and click Install. After installation, click Activate. The LW MWP Tools menu will appear in the left navigation menu.
